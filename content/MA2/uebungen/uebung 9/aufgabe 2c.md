$$
f_3(x) = \frac{
	\sin \left( \sqrt x \right)
}{
	1 + \cos \left( \sqrt x \right)
}
$$

# Ableitung

$$
\begin{align*}
	\frac{d}{dx} f_3 &= \frac{
		\frac{d}{dx} \left(
			\sin \left(
				\sqrt x
			\right)
		\right) \cdot \left(
			1 + \cos \left(
				\sqrt x
			\right)
		\right) - \left(
			\sin \left(
				\sqrt x
			\right)
		\right) \cdot \frac{d}{dx} \left(
			1 + \cos \left(
				\sqrt x
			\right)
		\right)
	}{
		\left(
			1 + \cos \left(
				\sqrt x
			\right)
		\right)^2
	} \\
\end{align*}
$$
